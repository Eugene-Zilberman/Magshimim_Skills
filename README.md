# ai-skills

> Подборка проверенных промптов и инструкций для AI-моделей для кода, текста и автономных агентов.

![открыто для контрибьюций](https://img.shields.io/badge/открыто%20для-контрибьюций-7F77DD) ![независимо от LLM](https://img.shields.io/badge/LLM-agnostic-gray) ![MIT лицензия](https://img.shields.io/badge/лицензия-MIT-0F6E56)

---

## Что это такое

**Скилл** - это структурированный, переиспользуемый набор инструкций, который говорит AI-модели, как именно подходить к конкретной задаче. 

### Категории

| Папка | Содержимое |
|---|---|
| `coding/` | Ревью кода, рефакторинг, генерация тестов, дебаггинг, commit-сообщения |
| `writing/` | Очеловечивание AI-текста, подбор тона, перевод, саммаризация |
| `agents/` | Исследование, планирование, анализ данных, автономное выполнение задач |
| `templates/` | Шаблон скилла + гайд по добавлению своих |

---

## Как использовать

1. **Найдите** - откройте нужную категорию и выберите скилл
2. **Скопируйте** - возьмите промпт из блока `## Prompt`
3. **Используйте** - вставьте как системный промпт или инлайн-инструкцию
4. **Улучшите** - доработали? Открывайте merge request с обновлённой версией

---

## Формат скилла

Каждый файл скилла имеет следующую структуру:

```markdown
# Название скилла
**Category:** coding / writing / agents
**Works with:** Claude / GPT / any LLM
**Author:** github handle
**Last tested:** YYYY-MM

## Purpose
Одно предложение → что делает этот скилл.

## Prompt
[ сам промпт ]

## Example
Вход → Выход

## Notes
Нюансы, ограничения, особенности конкретных моделей.
```

## Избранные скиллы

### `writing` → humanize-text
Переписывает AI-сгенерированный текст так, чтобы он проходил детекцию и читался естественно — корректирует ритм, словарный запас и разнообразие предложений.

### `coding` → code-review
Структурированное ревью: корректность, граничные случаи, производительность и стиль — с инлайн-комментариями.

### `coding` → write-tests
Генерирует полноценные юнит-тесты с граничными случаями для любой функции или модуля.

### `agent` → researcher
Агент для глубокого исследования: разбивает тему на подвопросы, опрашивает источники и возвращает структурированный итог с цитатами.

## Как контрибьютить

**Добавить новый скилл** - скопируйте `templates/SKILL_TEMPLATE.md`, заполните и откройте PR в нужную папку.

**Улучшить существующий** - нашли версию лучше? Обновите промпт + пример и обновите дату в поле *Last tested*.

**Добавить внешние источники** - нашли хороший скилл в интернете? Добавьте ссылку в `meta/sources.md` с коротким описанием.

**Сообщить о сломанном скилле** - откройте issue с указанием модели и даты. Промпты устаревают вместе с обновлениями моделей.
